Exemplo do Box Model

O Box Model é um conceito fundamental no CSS que descreve como os elementos HTML são representados como caixas retangulares.

Este é um exemplo de uma caixa com:
  • Margem: Espaço fora da borda.
  • Borda: A borda ao redor do conteúdo.
  • Preenchimento (Padding): Espaço entre o conteúdo e a borda.
  • Conteúdo: O próprio conteúdo dentro da caixa.
